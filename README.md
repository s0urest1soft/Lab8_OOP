This simple implementation allows adding and removing property owners. Owners can have several types of property, such as a car, an apartment, or a country house. For each property, two attributes are set: an attribute inherent only to this type of property and the price of the property. For each property, tax is calculated based on the price of that property. Functionality for working with JSON files is included: by specifying your own JSON file, you can retrieve information about the owner and their property, and you can also export information about any owner to a JSON file to find out about the taxes on their property. The program is a laboratory work for Object-Oriented Programming in C++.
